Welcome to Our Responsive Site!

This page demonstrates a fully responsive layout using CSS Grid and Tailwind CSS. Resize your browser window to see how the layout adapts from mobile to desktop sizes.

Key Features:

  • Generic Header with Site Title and Mobile Toggle.
  • Left Navigation (Categories, Quick Links).
  • Main Content Area for your primary information.
  • Right Navigation/Sidebar for related content or ads.
  • Generic Footer with copyright info.
  • Vanilla JavaScript for mobile menu toggle.

Responsive Behavior:

On mobile and tablet (below 1024px), all sections stack vertically. The left navigation is hidden by default and can be toggled open using the menu icon in the header.

On desktop (1024px and above), the layout transforms into a three-column structure: Left Nav, Main Content, and Right Nav, flanked by the full-width Header and Footer.

The columns are defined with specific widths for the sidebars (200px) and a flexible width for the main content (`1fr`), ensuring the main content always takes the remaining available space.
